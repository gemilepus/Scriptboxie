### ⭐Loop
> Repeatedly execute

* Action
  * @Times
  * Empty
> Times: run several times
> Empty: always running

* Example
  * Loop `5`

***

### ⭐Jump 
>Jump number of line

* Action
  * @Number
> Number: number of line

* Example
  * Jump `-3`

***

### ⭐Goto 
>Go to number of line

* Action
  * @Number
> Number: number of line

* Example
  * Goto `20`

***

### ⭐Run .exe 
>

* Action
  * @Path
> Path: .exe Path

* Example
  * Run .exe `C:\Users\Metro.exe`
  * Run .exe `C:\Program Files (x86)\......\chrome.exe -incognito`
  * Run .exe `CMD.exe /c C:\test.bat`
